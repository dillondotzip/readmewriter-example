# readmewriter-example

## Getting Started

### Installation

```bash
bun install
```

### Usage

#### How to start the dev server

```bash
bun run dev
```

#### How to build the project

```bash
bun run build
```

## API Documentation

| Method | Endpoint | Parameters | Description                               |
| ------ | -------- | ---------- | ----------------------------------------- |
| GET    | /        | None       | Returns 'Hello Hono!'                     |
| GET    | /random  | None       | Returns a random number between 1 and 100 |
