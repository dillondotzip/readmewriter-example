To install dependencies:

```sh
bun install
```

To run:

```sh
bun run dev
```

open http://localhost:3000

| /           | GET                                |
| ----------- | ---------------------------------- |
| description | Returns a simple greeting message. |

| /random     | GET                                         |
| ----------- | ------------------------------------------- |
| description | Returns a random integer between 1 and 100. |
